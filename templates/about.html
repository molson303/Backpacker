About Backpacker

Backpacker is an interactive shopping cart that was built using AngularJS and Bootstrap.

Functions of Backpacker

  • Information for all the products is pulled from the main_controller.js and is appended dynamically.
  • User is able to search by name using the Search Bar which is located in the navigation bar.
  • User is able to sort products according to price. Highest to Lowest or vice versa.
  • User is able to add products to the shopping cart
  • If a user decides they want more/less than what is selected they can use the dropdown to select a greater or lesser quantity
  • User is able to search by category
  • The checkout button at the top of the page adjusts when more items are added to the cart
  • When the user clicks the Checkout button they are taken to the shopping cart
